# Roadmap del Proyecto

# Fase 1 — Frontend E-commerce
Objetivo:
Construcción del frontend moderno del e-commerce.

Incluye:
- landing page
- catálogo
- navbar
- footer
- carrito
- login/register
- dashboard usuario

---

# Fase 2 — Backend y Base de Datos
Objetivo:
Construcción de APIs y persistencia de datos.

Incluye:
- autenticación
- productos
- pedidos
- usuarios
- pagos

Tecnologías:
- FastAPI
- PostgreSQL
- SQLAlchemy
- Docker

---

# Fase 3 — Integración Completa
Objetivo:
Conectar frontend y backend.

Incluye:
- consumo de APIs
- autenticación JWT
- carrito persistente
- pedidos reales

---

# Fase 4 — Machine Learning
Objetivo:
Implementar predicción inteligente de tallas.

Incluye:
- captura de datos físicos
- procesamiento de medidas
- recomendación de talla
- modelos ML

Tecnologías:
- PyTorch
- Scikit-learn
- Pandas

---

# Fase 5 — Visión Computacional
Objetivo:
Implementar detección corporal.

Incluye:
- landmarks corporales
- extracción de medidas
- análisis de postura

Tecnologías:
- MediaPipe
- OpenCV

---

# Fase 6 — Probador Virtual
Objetivo:
Implementar visualización virtual de prendas.

Incluye:
- overlay de ropa
- segmentación corporal
- render virtual

---

# Fase 7 — Producción
Objetivo:
Despliegue profesional del sistema.

Incluye:
- Docker
- CI/CD
- monitoreo
- seguridad
- optimización
